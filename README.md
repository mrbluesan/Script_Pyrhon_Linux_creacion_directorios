# Script
Script en Python para creación de directorios Nmap, docs y otros
Es importante mantener un orden en nuestra forma de trabajar con estructuras claras y para iniciar una auditoria considero que la creación de directorios es una constante y una tendencia, por ello existe este script en Python3 en Linux con el fin de crear los directorios necesarios haciendo ejecución en el lugar en que estemos posicionados en la terminal.

Pasos:
  copiar o descargar Script_directorios.py
  editar si es necesario para personalizar el codigo
  ejcutar por terminal en el lugar donde desees crear los directorios
